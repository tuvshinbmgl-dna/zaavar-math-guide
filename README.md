# Заавар (Math Guide)

Монгол ахлах ангийн **Математик 10·11·12** өөрөө сурах платформын ажиллах загвар (prototype).
Үндэсний сурах бичигт (econtent.edu.mn / БШУЯ) суурилсан, "сурах бичгээ ганцаараа сурахад"
зориулсан 4 чадвартай:

1. **Заавар (Guide)** — сурах бичгийг алхам алхмаар сургах: эхлэхийн өмнө оролд → ойлголт →
   бодсон жишээ (алхам бүрийг дараалан нээх) → бие даан бодох (сэжүүртэй) → эзэмшлийн шалгалт (80+ нээгдэнэ).
2. **Оношлох (Assessment)** — богино адаптив тест → чадварын зураглал (0–100) + хувийн төлөвлөгөө.
3. **AI багш (Tutor)** — Сократын аргаар, хариултыг шууд хэлэхгүй; **аль хичээлийн аль хэсгийг
   үзэхийг** яг хэлж, дутуу мэдлэгийн хувьд өмнөх ангийн хичээл рүү буцаана.
4. **Цоорхой нөхөх (Gap-fill)** — урьдчилсан нөхцөлийн граф + оношлох тест.

## Ажиллуулах

```bash
cd math-guide
./run.sh                 # Flask суулгаад http://127.0.0.1:5001 дээр ажиллуулна
# эсвэл:
python3 app.py
```

Browser дээр **http://127.0.0.1:5001** нээнэ үү.

### AI багшийг идэвхжүүлэх

AI tutor + question→lesson routing нь Anthropic Claude API ашигладаг. Өөрийн түлхүүрээ тохируул:

```bash
export ANTHROPIC_API_KEY=sk-ant-...     # эсвэл .env файлд хийнэ (.env.example-г хар)
./run.sh
```

- **Тутор чат** → `claude-sonnet-4-6` (стримлэг, Сократын).
- **Асуулт→хичээл чиглүүлэлт** → `claude-haiku-4-5` (хурдан, бүтэцлэг JSON).
- Түлхүүргүй үед **Заавар ба Оношлох бүрэн ажиллана**; зөвхөн AI чат идэвхгүй болж эелдэг мэдэгдэл харуулна.

## Бүтэц

```
math-guide/
├── app.py              # Flask routes + API (диагностик, тутор-стрим, чиглүүлэлт)
├── claude_client.py    # Claude API (stdlib urllib; Sonnet тутор + Haiku router)
├── store.py            # Контент ачаалагч + адаптив диагностик хөдөлгүүр
├── data/
│   ├── curriculum.json     # 3 ангийн бүтэн агуулга (навигац)
│   ├── knowledge_graph.json# 16 чадвар, урьдчилсан-нөхцөлийн ирмэгүүд
│   ├── diagnostic.json     # 33 даалгаврын адаптив сан (16 чадвар)
│   └── lessons/*.json      # 12 хичээл (9 уламжлал + 3 урьдчилсан)
```

Сурагчийн ахиц **browser-ийн localStorage**-д хадгалагдана (бүртгэл, DB шаардлагагүй).

## Агуулгын хамрах хүрээ (загвар)

- **Гүнзгий зүсэлт:** 11-р анги, Бүлэг VIII **Уламжлал** (8.1–8.9, 9 хичээл) — бүрэн заавар +
  бодлого + эзэмшлийн шалгалт.
- **Урьдчилсан давталт:** Шулууны налуу (10-р, 4.2), Функц (11-р, 3.1), Зэргийн функц (11-р, 3.2).
- **Навигацийн араг яс:** Математик 10/11/12-ийн бүх бүлэг, дэд хичээлийн жагсаалт.

## Гарал үүсэл

Дэлхийн шилдэг 5 платформоос санаа авсан (deep-research-ээр баталгаажуулсан):
ALEKS (мэдлэгийн орон зай → дараа юу үзэх), Khanmigo (хариулт нуудаг Сократ тутор),
IXL (0–100 эзэмшил + сурах бичигт холбосон төлөвлөгөө), Brilliant (эхлэхээс өмнө оролд),
Squilliam/Squirrel AI (нарийвчилсан чадварын граф).

## ⚠️ Эрх зүйн анхааруулга

Сурах бичгийн агуулга нь **БШУЯ-ны зохиогчийн эрхтэй** ("борлуулахыг хориглоно"). Энэ загвар нь
сурахад туслах зорилготой бөгөөд эх сурах бичиг рүү шууд холбоос (econtent.edu.mn) өгдөг.
**Нийтэд гаргахаас өмнө** агуулгыг дахин түгээх зөвшөөрлийг Яамнаас (textbook@mecs.gov.mn) авах
шаардлагатай.
